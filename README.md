🤖 AI Knowledge Hub
AI Knowledge Hub is a modern, interactive web application that displays random Artificial Intelligence facts inside a smooth, chat‑style interface.
It’s built using HTML, CSS, and JavaScript, and includes features like:

🌗 Dark/Light Mode Toggle

💬 Chat‑style message animations

🔊 Click sound effects

⚡ Random fact generator

🌐 Optional API integration for real‑time facts

🎨 Clean, modern UI with smooth transitions

This project is designed to showcase front‑end development skills while keeping the experience fun, interactive, and visually appealing.

📁 Project Structure
Code
├── index.html      # Main UI structure
├── style.css       # Styling, themes, animations
├── script.js       # Logic for facts, chat messages, theme toggle, sounds
└── sounds/         # Click sound effects
🧠 How It Works
script.js
Stores a list of AI facts

Generates a random fact on button click

Adds messages dynamically to the chat container

Plays a click sound

Toggles between dark and light themes

Includes optional API support for live facts

style.css
Defines dark/light theme variables

Styles the chat interface

Adds smooth animations

Creates a clean, modern layout

index.html
Builds the page structure

Loads the stylesheet and script

Contains the chat container, header, and buttons
